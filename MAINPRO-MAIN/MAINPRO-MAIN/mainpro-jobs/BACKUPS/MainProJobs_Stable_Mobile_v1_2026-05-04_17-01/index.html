Keyboard & gestures

From the main list, when the focus is not in a text field:

  • / — focus job search (same as the / chip in the field)
  • [ / ] — switch Active / History (when not typing in a field)
  • Clear filters in the list header — reset search, engineer, “My jobs”, and KPI/History filters in one go
  • ? or Settings → How it works — open this panel
  • Esc — close onboarding, full job, Park dialog, or photo; in search, clear text or close the keyboard if empty
  • Ctrl + S or ⌘ + S — save an engineer note (with the note field focused)
  • Narrow screen: pull down at the very top to refresh the list
  • Install the app (browser menu → “Install” or “Add to Home Screen”) to open it like an app. After the first full load, core files are cached for offline use (HTTPS or localhost only).
